# focusmaxxing

A Claude Code skill for a 4-step focus ritual: **meditation → brain dump + prioritization → work → closing**. For confronting tough work with peace instead of grinding.

## Install

Requires [Claude Code](https://claude.com/claude-code) (CLI, Desktop, or IDE extension — not the web app or mobile).

```
/plugin marketplace add paulkernfeld/focusmaxxing
/plugin install focusmaxxing@focusmaxxing
```

## Use

Start a session by saying "let's focusmaxx" in any Claude Code conversation. The skill walks you through the four steps. You drive transitions; Claude holds the structure.

Total time: ~1-2 hours.
